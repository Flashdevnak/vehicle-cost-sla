ขอรถเสริมหลังเส้นหลักพ่วงสาขาออกแล้ว

การควบคุมรถเสริม

1,000 บาท/ครั้ง

ลดการใช้รถเสริมเกินความจำเป็น และควบคุมให้การขอรถสอดคล้องกับปริมาณพัสดุจริง

เงื่อนไขที่ถูกตรวจ

  • เส้นหลักพ่วงสาขา A+B ออกเดินรถแล้ว แต่ยังมีการขอรถเสริมเพิ่ม
  • ปริมาณพัสดุของจุด A/B ที่โหลดบนรถพ่วงสาขาน้อยกว่า 40%

ยกเว้น / อุทธรณ์

  • รถเที่ยวหลักบรรทุกเต็ม และเส้นหลักพ่วง A+B ก็บรรทุกเต็ม
  • ปริมาณพัสดุของจุด A/B บนรถพ่วงตั้งแต่ 40% ขึ้นไป
  • มีข้อมูลจำนวนพัสดุและรูปการโหลดประกอบครบถ้วน

หลักฐานที่ควรเตรียม

  • จำนวนพัสดุของรถเที่ยวหลัก
  • จำนวนพัสดุของเส้นหลักพ่วง A+B
  • รูปถ่ายการโหลด/สภาพการโหลด
  • ข้อมูลการขอรถเสริมและเวลาปล่อยรถ
2.1

ปล่อยรถล่าช้ากว่ากำหนด

การควบคุมอัตราออกเดินรถตรงเวลา

300 บาท/ครั้ง

ควบคุมเวลาปล่อยรถ FD + LH ทั้งรถเที่ยวหลักและรถเสริม เพื่อไม่ให้กระทบ SLA ปลายทาง

เงื่อนไขที่ถูกตรวจ

  • รถ FD + LH เที่ยวหลักหรือเที่ยวเสริมออกเดินรถล่าช้ากว่ากำหนด

ยกเว้น / อุทธรณ์

  • กรณีรถ Standby มาถึงล่าช้า สามารถยื่นอุทธรณ์เพื่อขอยกเว้นได้
  • ต้องมีข้อมูลเวลารถ Standby และหลักฐานประกอบชัดเจน

หลักฐานที่ควรเตรียม

  • เวลา Standby ถึงหน้างาน
  • เวลาเริ่มโหลดและเวลาปล่อยรถ
  • ข้อมูลจากระบบ/หน้างานที่ตรวจสอบได้
2.2

ปล่อยรถหลังเวลา 12:00 น.

การควบคุมอัตราออกเดินรถตรงเวลา

300 บาท/ครั้ง

ควบคุมไม่ให้รถ FD + LH ออกหลังเที่ยง โดยเฉพาะรอบที่กระทบการส่งต่อและ SLA

เงื่อนไขที่ถูกตรวจ

  • รถ FD + LH เที่ยวหลักหรือเที่ยวเสริมออกเดินรถหลังเวลา 12:00 น.

ยกเว้น / อุทธรณ์

  • กรณีล่าช้าจากรถ Standby สามารถยื่นอุทธรณ์ได้
  • กรณีนี้ไม่คิดซ้ำกับหัวข้อรถออกล่าช้า โดยนับเพียง 1 ครั้งตามข้อมูลตรวจพบ

หลักฐานที่ควรเตรียม

  • เวลา Standby
  • เวลาออกเดินรถจริง
  • ข้อมูลเหตุล่าช้าและผู้รับผิดชอบหน้างาน

รถเที่ยวหลักบรรทุกไม่เต็มคัน และยังมีพัสดุคงคลัง

การควบคุมอัตราบรรทุก

500 บาท/ครั้ง

เพิ่มประสิทธิภาพการใช้รถเที่ยวหลัก ไม่ปล่อยรถไม่เต็มในขณะที่จุดดรอปยังมีพัสดุเหลืออยู่

เงื่อนไขที่ถูกตรวจ

  • รถเที่ยวหลักคันสุดท้ายของเส้นทางออกเดินรถโดยบรรทุกไม่เต็ม
  • ยังมีพัสดุตกค้างอยู่ที่จุดดรอปของเส้นทางนั้น

ยกเว้น / อุทธรณ์

  • ต้องมีข้อมูลจำนวนพัสดุที่โหลดจริง
  • ต้องมีรูปการโหลดสินค้า
  • ต้องมีภาพ CCTV แบบ Real-time ของพัสดุคงค้างที่จุดดรอป

หลักฐานที่ควรเตรียม

  • จำนวนพัสดุโหลดจริง
  • รูปถ่ายการโหลดท้ายรถ/ภายในรถ
  • ภาพ CCTV จุดดรอปแบบ Real-time
  • ยอดพัสดุคงคลังของเส้นทาง
4

เคลียร์เครื่องคัดแยกไม่ทันก่อนเวลาปิดรอบรถ

การควบคุม SLA คงคลัง

0.5 บาท/ชิ้น

เงื่อนไขที่ถูกตรวจ

  • ก่อนเวลาปล่อยรถ ยังเคลียร์พัสดุบนเครื่องคัดแยกไม่เรียบร้อย
  • ทำให้พัสดุขนาดเล็กล่าช้าหรือออกไม่ทันรอบ

ยกเว้น / อุทธรณ์

  • พิจารณาจากข้อมูลระบบและหลักฐานหน้างาน
  • ต้องระบุจำนวนพัสดุที่ได้รับผลกระทบได้ชัดเจน

หลักฐานที่ควรเตรียม

  • จำนวนพัสดุที่ล่าช้า
  • เวลาเคลียร์เครื่องคัดแยก
  • หลักฐานจากระบบ/ภาพหน้างาน

  • ตำแหน่งอื่นเชื่อมโยงตามขอบเขตประเทศ/สายงานที่กำหนด

เช็กลิสต์ควบคุมหน้างาน

ใช้ตรวจสอบก่อนปิดรอบและก่อนยื่นอุทธรณ์

ก่อนขอรถเสริม

  • ตรวจปริมาณพัสดุของเส้นหลักและจุดพ่วง
  • ตรวจอัตราโหลดว่าถึงเกณฑ์ 40% หรือไม่
  • บันทึกรูปโหลดและจำนวนพัสดุก่อนส่งคำขอ

ก่อนปล่อยรถ

  • ตรวจเวลา Standby และเวลาโหลดจริง
  • ตรวจว่ารถไม่ออกช้าหรือออกหลัง 12:00 น. โดยไม่มีเหตุรองรับ
  • บันทึกข้อมูลกรณีผิดปกติทันที

ก่อนปิดรอบ

  • ตรวจพัสดุคงคลังที่จุดดรอป
  • ตรวจการเคลียร์เครื่องคัดแยก
  • ตรวจภาพ CCTV/รูปหน้างานสำหรับเคสเสี่ยง

ก่อนยื่นอุทธรณ์

  • รวบรวมข้อมูลโหลด, รูปถ่าย, CCTV และเวลาที่เกี่ยวข้อง
  • แยกเหตุ Standby ช้าออกจากเหตุหน้างาน
  • สรุปข้อมูลส่งผู้ตรวจสอบตามช่องทางที่กำหนด
